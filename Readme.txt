# JDBC Connectivity Project

This project demonstrates JDBC connectivity in Java, providing functionalities for inserting, updating, deleting, fetching, and retrieving all data from an SQL database. It includes examples of using `addBatch()` and `executeBatch()` for efficient batch processing.

## Features

- **Insertion**: Add new records to the database.
- **Updation**: Update existing records based on specific criteria.
- **Deletion**: Remove records from the database.
- **Fetch**: Retrieve specific records based on a query.
- **Retrieve All**: Fetch all records from a table.
- **Batch Processing**: Efficiently handle multiple operations using `addBatch()` and `executeBatch()`.

## Prerequisites

- **Java Development Kit (JDK)**
- **JDBC Driver** for MySQL database 
- **SQL Database** setup

## Technologies Used

- **Java**
- **JDBC**
- **SQL Database** (MySQL)
- **Batch Processing**
